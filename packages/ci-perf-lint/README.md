# ci-perf-lint

This package is the unscoped CLI entrypoint for `@yoshi-taka/ci-perf-lint`.

For documentation, usage examples, and rule docs, see:

- https://ci-perf-lint.veritycost.com/
- https://github.com/yoshi-taka/ci-perf-lint

Quick start:

```sh
bunx ci-perf-lint .
# or
npx ci-perf-lint .
```
